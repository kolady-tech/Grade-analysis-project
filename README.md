# Salary Analysis Project

## Description
This project analyzes a dataset containing employee salary data.
It performs:
- Data cleaning
- Duplicate removal
- Null handling
- Feature engineering
- Data visualization
- Pattern analysis

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Outputs
- Cleaned dataset
- Visual charts
- Analytical insights

## Learning Outcome
Built a complete data pipeline from raw CSV to insights.
